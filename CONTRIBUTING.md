# Contributing

```bash
pnpm install    # Install dependencies
pnpm dev        # Start development
pnpm test       # Run tests
```

## Pull Requests

1. Fork and clone
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes and test
4. Add a changeset: `pnpm changeset`
5. Commit with [semantic messages](https://gist.github.com/joshbuchea/6f47e86d2510bce28f8e7f42ae84c716)
6. Open a PR

## Changesets

Run `pnpm changeset` before committing to describe your changes. Select patch/minor/major based on impact.

## Issues

Report bugs at [GitHub Issues](https://github.com/NEARBuilders/near-intents-cli/issues).
